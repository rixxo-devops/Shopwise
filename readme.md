# Shopwise App Documentation

## What is Shopwise?

Shopwise is a powerful integration app that connects your Shopify store with Orderwise, enabling:

- **Order Synchronization** - Automatically send orders from Shopify to Orderwise
- **Inventory Management** - Keep stock levels synchronized between platforms
- **Status Updates** - Track order status changes across both systems
- **Product Mapping** - Sync product data and attributes
- **Payment & Shipping Mapping** - Ensure proper order processing
