## 2. Typography

### Primary Font Family
**Poppins Bold** - Used for all headings (H1, H2, H3, H4, H5, H6)

**Specifications:**
- Weight: Bold
- Letter case: Sentence case
- Line height: 110% of font size
- Letter spacing: -1.2px

### Secondary Font
**Roboto Regular** - Used for body copy

**Specifications:**
- Weight: Regular
- Letter case: Sentence case
- Line height: 150% of font size
- Font size: 18.4% of the main heading on the design (e.g., H1 tag on webpage)
- Minimum digital font size: 10px
- Minimum print font size: 8pt
- 6pt allowed for T&Cs or captions

### Web Typography Sizes

**Desktop:**
- H1: 3 rem
- H2: 2.4 rem
- H3: 2 rem

**Mobile:**
- H1: 1.44 rem
- H2: 1.2 rem

### Font Alignment
Always left-align text (never center align)

## 3. Logo

### Logo Colors
- **Primary**: The Rixxo logo is always presented in **Green** (#33CC99)
- **On colored backgrounds**: Prefer Green, but use greyscale if Green doesn't work
- **Never use**: The logo should never be presented in other primary colors (Blue, Red, Pink, Orange, Yellow)

### Logo Files
SVG and color formats available in Illustrator format

### Favicon Sizes
- File size: 512 x 512 px
- 16x16 px: Browser tabs and bookmarks
- 32x32 px: High-resolution displays and Windows taskbar
- 48x48 px: Various platforms, recommended by Google
- 96x96 px: Higher resolution displays
- 512x512 px: PWAs (largest size)

## 4. Other Styling Elements

### Design Patterns

**Gradients:**
- Use only for backgrounds (websites, slides, print)
- Never for text, logos, icons, or borders
- Must flow diagonally from one corner to the opposite corner
- Light mode: Start 10% `#F9F4F4` → End 100% `#F9F4F4`
- Dark mode: Start 10% `#1A2F37` → End 100% `#4A7582`

**Light/Dark Mode:**
- Mixed usage allowed for digital assets and professional print
- Use light mode for office printing (save ink)
- Use dark mode for presentations/auditoriums (reduce eye strain)

### Button Styles

**Four Button Types:**
1. Solid
2. Stroked
3. Text-only
4. Icon-only

**Button Specifications:**
- Font size: Same as body copy in layout
- Top/bottom padding: 150% of font size (e.g., font 16px = 22px padding)
- Left/right padding: 200% of font size (e.g., font 18px = 36px padding)
- Stroke weight: 5% of button's text size (minimum 1px)
- Icon spacing: Equal to icon's width
- Icon height: Match text height

**Button Colors:**
- Can use any color from the primary palette
- Available in both light and dark mode
- Primary, Secondary, Tertiary, and Link styles

### Link Colors
- Links can use any color from the primary palette
- Text-only buttons with underline follow same color rules

### Title/Heading Highlighting
- Highlight up to 3 words per line in Rixxo Green (#33CC99)
- Use patterns like 2-1-2, 3-1-3, or 2-3 for visual balance
- Only highlight if core message is clear from highlighted words alone
- Never use colors other than Rixxo Green for highlighting

### Punctuation Guidelines

**Short, fragment-style headings:**
- No punctuation (except question marks)
- Cleaner design, more impact

**Headlines with 5+ words:**
- Use punctuation intentionally for clarity

**Marketing-style dual sentences:**
- Use punctuation (e.g., "Connect Orderwise. Automate the rest.")
- Emphasizes rhythm and brand tone

### Bullet Points
- Use sparingly for clarity
- Reserved for breaking down sub-points within a single idea
- Use separate lines for separate ideas
- Never use full stops in bullet points
- Never center-align bullet points
- Always left-align

## 5. Brand Voice & Tone

### Voice Characteristics
- **Minimalist**: Every word has a reason to be there
- **Confident**: No hedging, clear statements
- **Collaborative**: "We" language, team-focused
- **Progressive**: Always moving forward, always refining

### Writing Style
- Speak like a mentor, not an influencer
- Direct, grounded, and practical
- No ego, no performance
- Focus on outcomes, not features
- Use plain English, avoid jargon
- Get straight to the point
